Angular

Angular is an application design framework and development platform for creating efficient and sophisticated single-page apps.AngularJs is a Javascript Framework is created to build the dynamic web application (dynamic web app), it is often used to create a single page application (Single Page Application - SPA). And works based on extending more attributes (attributes) to HTML's tags (Angular rule attributes). AngularJS is a free platform and is preferred by developers around the world.

Node.JS

Node.Js is a productive, battle-tested JavaScript framework for building modern web applications. It includes everything you need to build rich UIs that work on any device. Nodejs is a platform independent development (Platform) built on top of Chrome's Javascript Runtime with which we can build network applications quickly and easily.
